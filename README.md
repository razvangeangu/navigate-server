
# Navigate Server

This is a server that proves indoor navigation capabilities for an iOS device. The research has been conducted by Răzvan Geangu supervised by [Dr. Andrew Coles](https://nms.kcl.ac.uk/andrew.coles/) | Department of Informatics, King’s College London.

Please refer to the [main repository](https://github.com/razvangeangu/navigate-ios) for more information.
